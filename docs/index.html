Sizuka Project

Introduction

Sizuka Project provides you a solution to present your static resources stored on Aliyun OSS.

Sizuka is a simple and quick solution for open and even half open buckets. It uses tree view to display the objects and has features to let users preview, share and open directly, and also, it checks the file type and gives correct Content Type.

Sizuka Player

For the audio record files stored in Aliyun OSS and maintained by Sizuka Service, you can use Sizuka Player as a client on iOS devices.

PRIVACY DECALARATION

ALL DATA STORED ON THY ALIYUN OSS COUND BE ACCESSED BY THE DEPLOYED (I.E. PRIVILEGE GRANTED) SIZUKA SERVICE AND CLIENT AS THE KEY FUNCTION PROVIDED. BUT IT DOES NOT MEAN THAT MAY BE ACCESSED BY THE SIZUKA PROJECTS.
